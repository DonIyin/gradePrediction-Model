# 🎓 Student Grade Prediction System

A machine learning web application that predicts whether a secondary school student will **pass or fail** their final Mathematics or Portuguese exam — based on demographic, family, and lifestyle attributes, **without relying on prior grade records**.

Built with Python, Scikit-learn, XGBoost, and Streamlit.

---

## 📌 What This Project Does

Most grade prediction systems use previous grades (G1, G2) to predict the final grade (G3). That's easy — but not useful. A teacher already knows if a student is struggling by the time they have two period grades.

This system deliberately **excludes G1 and G2**. The model works from background and lifestyle data that exists at the **start of term** — enabling early identification of at-risk students before any academic failure occurs.

## 📊 Dataset

**Source:** UCI Student Performance Dataset, available on [Kaggle](https://www.kaggle.com/datasets/dskagglemt/student-performance-data-set)

**Citation:** Cortez, P. & Silva, A. (2008). *Using Data Mining to Predict Secondary School Student Performance.* EUROSIS.

| Property | Value |
|---|---|
| Students (Math) | 395 |
| Students (Portuguese) | 649 |
| Features used | 30 |
| Target | G3 (0–20) → encoded as Pass (≥10) / Fail (<10) |
| Missing values | None |

**Feature categories:** demographics, parents' background, school choice, time and logistics, academic history, social and lifestyle habits.

> G1 and G2 (period grades) are intentionally excluded so the model is useful at the start of term.

---

## 🤖 Models Trained

Six classifiers are trained on each dataset:

| Model | Description |
|---|---|
| Logistic Regression | Linear classifier — fast, interpretable, strong baseline |
| Random Forest | Ensemble of 100 decision trees — handles non-linear relationships |
| XGBoost | Gradient boosting — often highest accuracy, tuned with GridSearchCV |
| SVM | Support Vector Classifier — effective in high-dimensional spaces |
| KNN | K-Nearest Neighbours — predicts based on the 5 most similar students |
| Decision Tree | Single tree — fast and interpretable but prone to overfitting |

Random Forest and XGBoost are additionally tuned using **GridSearchCV** (5-fold cross-validation) to find optimal hyperparameters.

## 🔬 Methodology

### 1. Target Engineering
```python
df['pass'] = df['G3'].apply(lambda x: 1 if x >= 10 else 0)
```
G3 (0–20 scale) is converted to a binary label. 10/20 is the standard pass mark.

### 2. Preprocessing
- Drop G1, G2, G3 columns
- Label-encode all categorical columns (alphabetical integer mapping)
- Scale features to mean=0, std=1 using StandardScaler

### 3. Training Strategy
- 80/20 stratified train-test split (`stratify=y` preserves class balance)
- 5-fold Stratified Cross-Validation for robust performance estimation
- GridSearchCV hyperparameter tuning for Random Forest and XGBoost

### 4. Evaluation Metrics
- **Accuracy** — overall correctness
- **Precision** — of predicted Pass, how many actually passed?
- **Recall** — of actual Pass, how many did we catch? *(most important metric here)*
- **F1 Score** — harmonic mean of precision and recall
- **AUC-ROC** — performance across all decision thresholds

**Recall is prioritised** because a false negative (missing a student who will fail) is more costly than a false alarm.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/DonIyin/student-grade-prediction
cd student-grade-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download the dataset
Download `student-mat.csv` and `student-por.csv` from [Kaggle](https://www.kaggle.com/datasets/dskagglemt/student-performance-data-set) and place both files in the `data/` folder.

### 4. Run the notebook
Open `notebook/model.ipynb` and run all cells from top to bottom.

This trains all 6 models on both datasets, runs hyperparameter tuning, and saves 14 `.pkl` files to the `models/` folder. **The app cannot run without these files.**

### 5. Launch the app
```bash
streamlit run app.py
```

The app opens at `http://localhost:8501`.
